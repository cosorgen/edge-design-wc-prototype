# Edge Design Web Component Prototype

## Setup

You need to add a .env file to the root of the server folder with the following:

```bash
BING_APP_ID="ID"
WEATHER_APP_ID="ID"
```

Then run the following

```bash
npm install
npm run dev:server
npm run dev:client
npm run serve
```

Then navigate to http://localhost:4000 to see your work
